# 【整合資產管理平台】規格計劃書

# 1. 專案概述

## 1.1 專案背景與目的

大多數人的資產散落在不同機構：本土銀行、外幣帳戶、2-3 個股票券商、多個加密貨幣交易所。沒有辦法一目了然看到自己的整體淨值。本專案打造一個一站整合、各帳戶統一檢視的資產管理平台，類似於「終極版」的網路銀行 + 投資組合管理工具。
- 核心價值：讓使用者 30 秒內了解自己的整體財務狀況
## 1.2 目標受眾（TA）

25-40 歲專業人士 — 多帳戶分散，想要統一檢視
投資者（股票/ETF）— 需要一眼看到所有倉位與績效
加密貨幣持有者 — 各交易所餘額不同步，難以計算總資產
理財規劃顧問 — 需要快速掌握客戶整體資產配置
家庭財務管理者 — 共同帳戶多位成員，需要共享視圖
## 1.3 專案範圍

### In Scope（要做）

- 銀行帳戶連結（Open Banking API 或手動餘額輸入）
- 股票/ETF 帳戶連結（券商 API 或手動）
- 虛擬貨幣交易所連結（CEX API：Binance / Coinbase / MAX）
- 總資產儀表板（Dashboard）
- 資產配置視覺化（圓餅圖 / 長條圖）
- 淨值趨勢圖（時間序列）
- 單一帳戶詳情頁
- 使用者註冊 / 登入
- 即時價格更新（股票 + 加密貨幣）
### Out of Scope（不做）

- 交易執行（下單、轉帳）
- 信用卡管理
- 房產 / 汽車 / 其他非金融資產
- 稅務申報
- 真人理專顧問 / 機構版（僅個人用戶）
## 1.4 參考網站分析

- Personal Capital（personalcapital.com）— 優點：資產總覽完整、投資分析強；缺點：僅支援美國帳戶
- Mint（mint.com）— 優點：免費、介面簡單；缺點：無加密貨幣、廣告多
- Monarch Money（monarchmoney.com）— 優點：夫妻共享、現代 UI；缺點：訂閱制昂貴（US$99/年）
- Crypto.com DeFi Wallet — 優點：加密貨幣整合佳；缺點：僅加密資產、無傳統金融
- YNAB（ynab.com）— 優點：預算管理優秀；缺點：學習曲線高、非投資取向
切入點：結合 Personal Capital 的投資分析 + Mint 的免費簡單 + 加密原生支援，為中文用戶（台灣優先）打造的全方位資產管理工具。
# 2. 資訊架構與動線

## 2.1 網站地圖

- /（Landing Page）：價值主張、功能展示、安全說明（銀行級加密）、開始免費使用 CTA
- /login — 登入頁
- /register — 註冊頁（Email + 密碼，或 Google 登入）
- /dashboard — 會員首頁（總資產卡片、淨值趨勢圖、資產配置圓餅圖、帳戶餘額列表、近期異動）
- /accounts — 帳戶管理（新增帳戶向導、個別帳戶詳情、刪除/隱藏帳戶）
- /portfolio — 投資組合分析（圓餅圖+歷史變化、風險承受度、績效比較 vs 大盤）
- /transactions — 交易記錄（整合流水、篩選、每月收支分類）
- /settings — 個人資料、安全管理、通知設定、匯出資料、刪除帳號
- /help — 銀行連結教學、常見問題、聯絡客服
## 2.2 使用者動線

新訪客 → 瀏覽 Landing → 點擊「開始免費使用」→ 註冊流程（Email + 密碼）→ Email 驗證 → 首次引導新增第一個帳戶 → 選擇帳戶類型（銀行/股票/加密）→ 授權綁定或手動輸入 → 返回 Dashboard → 總資產 Dashboard 載入
常態使用：直接進 Dashboard → 顯示總資產 + 配置圖 → 可點擊帳戶進詳情 / 點擊 Portfolio 查看分析 / 點擊 Transactions 瀏覽流水 / 點擊 Settings 管理設定或新增帳戶
## 2.3 使用者旅程圖

首次註冊：瀏覽 Landing（4）→ Email 驗證（3）→ 新增第一個帳戶（5）→ 看到第一筆餘額同步（5）→ Dashboard 完整呈現（5）
陸續新增帳戶：新增股票券商（4）→ 新增加密貨幣交易所（4）→ 感到「終於知道自己的錢都在哪裡」（5）
日常檢視：每天打開看一眼總資產（4）→ 發現淨值趨勢圖往上（5）→ 收到大額異動通知（4）
進階分析：查看資產配置圓餅圖（4）→ 發現加密比重過高（4）→ 調整投資配置決策（5）
# 3. 視覺與使用者介面需求

## 3.1 品牌設計指南

- Primary: #10B981（財富成長綠，代表資產增長）
- Secondary: #3B82F6（信任藍，代表安全可靠）
- Accent: #F59E0B（比特幣橙，代表加密資產）
- Background: #F8FAFC（淺灰白，乾淨專業）
- Card BG: #FFFFFF / Text Primary: #1E293B / Text Secondary: #64748B
- Success: #10B981 / Danger: #EF4444（虧損/警示）
- 字體：Inter（數字英文）/ Noto Sans TC（中文）/ DM Mono（數字顯示）
## 3.2 跨裝置支援

- Desktop ≥ 1024px：三欄 Dashboard（總資產/配置圖/近期異動）
- Tablet 768-1023px：雙欄，圖表縮窄
- Mobile < 768px：單欄，總資產卡片置頂，圖表可滑動
# 4. 前端功能規格

## 4.1 Landing Page

- Hero：主標「掌控你的全部財富」/ 副標「銀行、股票、加密，一個平台就看清楚」/ CTA「免費開始」
- 安全承諾：銀行級 AES-256 加密、Read-only 存取、永不下單
- 功能展示：三欄（帳戶整合 / 即時同步 / 智能分析）
- 支援機構 Logo 牆：常見銀行 + 券商 + 交易所
- 見證牆（2-3 個使用者見證）、FAQ（Accordion）、Footer（隱私權政策/服務條款/聯絡）
## 4.2 會員系統

- 註冊：Email + 密碼（8+ 字元）/ Google OAuth
- 驗證：Email 驗證連結 / 登入：Email + 密碼 / Google OAuth
- 2FA：可選 TOTP（Google Authenticator）/ 密碼重設：Email 連結
## 4.3 Dashboard（總資產首頁）

- 總資產卡片：大字體顯示總額（NT$）+ 日/月/年變化%
- 淨值趨勢圖：折線圖，預設近 30 天，可切換 90天/1年/全部
- 資產配置圓餅圖：銀行/股票/加密三類分布，hover 顯示%
- 帳戶卡片列表：各帳戶餘額，點擊進入詳情
- 近期異動：最近 5 筆大額進帳/出帳
## 4.4 帳戶連結

- 銀行：Open Banking API（台灣：FutureVault / 咚咚支付）/ 手動餘額
- 股票：API（如券商支援）/ 手動（券商後台截圖上傳解析）
- 加密：交易所 API Read-only Key（Binance/Coinbase/MAX）
## 4.5 投資組合分析

- 資產配置圖：圓餅圖 + 長條圖，可按時間比較
- 風險分布：各帳戶/資產的風險係數（低/中/高）
- 績效對比：與大盤（0050/SPY/BTC）走勢對照
- 區塊鏈 Gas 追蹤：顯示平均 Gas Fee（以太坊用戶）
## 4.6 交易記錄

- 整合流水：自動匯集所有帳戶的交易
- 分類標籤：自動（薪轉/消費/股息/買幣/賣幣）
- 收支摘要：月支出/收入/儲蓄率 / 匯出：CSV/Excel 下載
